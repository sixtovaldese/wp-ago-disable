# aGo Disable

WordPress plugin: toggle dashboard to disable WordPress features you do not use — comments, Gutenberg, auto-updates, pingbacks, search, RSS, admin bloat. **Free, no Pro tier.**

[English](#english) · [Español](#espanol)

---

## English

aGo Disable gives you a single screen with 14 toggles to turn off WordPress features that many sites do not need. Safe defaults are enabled on first activation. The rest are off by default; flip them only when you understand the impact.

Every toggle ships with a 1-line summary plus an expandable "More info" block (how it works, why, caveats). No guesswork.

### Features

- 14 reversible switches across Content, Updates, Admin UX and System.
- Safe defaults seeded on first activation (attachment pages, author archives, pingbacks, admin notices, news widget, heartbeat throttle).
- Auto-save on toggle change.
- Sidebar with quick links (PageSpeed, GTmetrix, Sucuri SiteCheck), donation cards, cross-sell to other aGo Lab plugins.
- English, Spanish (es_ES) and Brazilian Portuguese (pt_BR) translations bundled.
- No external calls, no telemetry, no Pro upsell.

### Installation

1. Upload the `ago-disable` folder to `/wp-content/plugins/` or install via the Plugins screen.
2. Activate the plugin.
3. Go to **aGo Tools → Disable**. Safe defaults are already enabled; tune the rest.

## Español

aGo Disable te da una sola pantalla con 14 interruptores para desactivar funciones de WordPress que muchos sitios no necesitan. Los defaults seguros se activan en la primera instalación. El resto vienen apagados; activalos solo cuando entiendas el impacto.

Cada interruptor trae un resumen de 1 línea más un bloque "Más info" expansible (cómo funciona, por qué, precauciones). Sin adivinanzas.

### Funcionalidades

- 14 interruptores reversibles agrupados en Contenido, Actualizaciones, UX del admin y Sistema.
- Defaults seguros activados en la instalación (páginas de adjunto, archivos de autor, pingbacks, avisos del admin, widget de noticias, throttle de heartbeat).
- Auto-guardado al cambiar cualquier interruptor.
- Sidebar con enlaces útiles (PageSpeed, GTmetrix, Sucuri SiteCheck), tarjetas de donación y cross-sell a otros plugins de aGo Lab.
- Traducciones bundleadas: inglés, español (es_ES), portugués brasileño (pt_BR).
- Sin llamadas externas, sin telemetría, sin presión de Pro.

### Instalación

1. Subí la carpeta `ago-disable` a `/wp-content/plugins/` o instalá vía el dashboard de plugins.
2. Activá el plugin.
3. Andá a **aGo Tools → Disable**. Los defaults seguros ya están activos.
